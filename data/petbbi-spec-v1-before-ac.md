# Petbbi(펫삐) 기능 명세서 v1.0

**제품명**: Petbbi(펫삐) — AI 펫 캐릭터·일기 플랫폼
**작성일**: 2026-06-20
**작성자**: (주)엔트레씨 프로덕트팀
**버전**: 1.0
**대상 플랫폼**: iOS / Android 모바일 앱

> 본 문서는 출시 1차 범위(MVP)의 기능 명세서입니다. 반려동물 사진을 업로드하면 AI가 외형·성격 특징을 분석해 전용 캐릭터를 생성하고, 캐릭터가 자동으로 일기를 작성하며, 사용자들이 커뮤니티에서 공유하는 글로벌 펫 플랫폼입니다. 각 기능은 "정상 동작" 기준으로 기술되어 있으며, QA 팀은 이 문서를 입력으로 받아 테스트 케이스를 설계합니다.

---

## 1. 반려동물 프로필 등록

반려동물의 기본 정보와 대표 사진을 등록하여 AI 분석의 기반 데이터를 만드는 기능.

### 1-1. 기본 정보 입력

- 입력 항목: 반려동물 이름(필수), 종류(강아지/고양이/기타 중 선택, 필수), 품종(선택), 생년월일(선택), 성별(수컷/암컷/미상)
- 이름: 1자 이상 20자 이하, 이모지 입력 허용
- 생년월일: 캘린더에서 선택, 오늘 이전 날짜만 선택 가능
- 한 계정당 반려동물 프로필 최대 5마리까지 등록 가능
- 입력 완료 후 "다음" 버튼 → 대표 사진 업로드 화면으로 이동

### 1-2. 대표 사진 업로드

- 카메라 촬영 또는 갤러리에서 선택하여 대표 사진 1장 등록
- 지원 형식: JPG, PNG, HEIC
- 권장 해상도: 1:1 정사각형 비율, 최소 512×512px
- 업로드 후 미리보기 화면에서 확인 → "등록" 버튼으로 확정
- 등록된 대표 사진은 이후 AI 캐릭터 생성의 입력으로 사용됨
- 프로필 화면 상단에 대표 사진과 기본 정보가 함께 표시됨

### 1-3. 프로필 수정·삭제

- 등록한 반려동물 프로필의 기본 정보와 대표 사진을 언제든 수정 가능
- 프로필 삭제 시 해당 반려동물의 캐릭터·일기도 함께 삭제됨
- 프로필 목록에서 여러 마리 중 대표 반려동물 1마리를 지정할 수 있음

## 2. AI 캐릭터 생성

업로드된 반려동물 사진을 AI가 분석하여 외형·성격 특징이 반영된 전용 캐릭터를 생성하는 기능.

### 2-1. 캐릭터 생성 요청

- 프로필 화면에서 "AI 캐릭터 만들기" 버튼 클릭 → 생성 시작
- AI가 대표 사진의 외형(털색, 귀 모양, 체형 등)과 사용자가 입력한 성격 키워드(활발함/차분함/장난꾸러기 등 최대 3개 선택)를 분석
- 생성 중 로딩 화면에 진행률(%)과 예상 소요 시간 표시
- 평균 생성 소요 시간: 15~30초

### 2-2. 생성 결과 표시

- 생성 완료 시 캐릭터 이미지 3종(정면/측면/전신)을 카드 형태로 표시
- 사용자는 마음에 드는 캐릭터를 "내 캐릭터로 저장" 버튼으로 확정
- 저장된 캐릭터는 프로필과 일기, 커뮤니티 게시물에 사용됨
- 캐릭터에는 AI가 분석한 성격 요약 문구가 함께 표시됨 (예: "호기심 많은 모험가")

### 2-3. 재생성 한도

- 결과가 마음에 들지 않으면 "다시 생성" 버튼으로 재생성 가능
- 재생성은 1일 최대 3회까지 허용 (고비용 AI 연산 최적화 정책)
- 재생성 시 직전 결과와 다른 변형 캐릭터가 생성됨
- 남은 재생성 횟수가 버튼 옆에 표시됨

## 3. AI 일기 자동생성

생성된 캐릭터가 반려동물의 하루를 1인칭 시점의 일기로 자동 작성하는 기능.

### 3-1. 일기 생성 트리거

- 사용자가 사진을 추가 업로드하거나 활동을 기록하면 일기 생성이 트리거됨
- 하루 1회 자동 생성 (오후 9시 기준) 또는 사용자가 "오늘의 일기 쓰기" 버튼으로 수동 생성
- 생성 전 사용자가 오늘의 키워드(산책/간식/낮잠 등)를 선택할 수 있음

### 3-2. 일기 내용 생성

- 캐릭터의 성격과 선택된 키워드를 반영하여 1인칭 일기 텍스트 생성 (200~400자)
- 일기에는 캐릭터 이미지와 날짜가 함께 표시됨
- 생성된 일기는 자동 저장되며 "일기장" 탭에서 날짜별로 조회 가능
- 사용자가 일기 내용을 직접 수정·삭제 가능

### 3-3. 다국어 일기

- 일기는 사용자가 설정한 언어로 생성됨 (한국어/영어/일본어 지원)
- 설정 화면에서 일기 생성 언어를 변경할 수 있음
- 언어 변경 시 이후 생성되는 일기부터 적용됨

## 4. 커뮤니티 피드

사용자들이 캐릭터·일기 기반 콘텐츠를 공유하고 소통하는 소셜 기능.

### 4-1. 게시물 작성

- 캐릭터 이미지 또는 일기를 첨부하여 게시물 작성
- 텍스트 본문 최대 500자, 사진 최대 4장 첨부 가능
- 해시태그 입력 가능 (최대 10개)
- "게시" 버튼 클릭 시 피드에 즉시 노출

### 4-2. 피드 조회

- 최신순/인기순 정렬 전환 가능
- 게시물에 좋아요·댓글 작성 가능
- 무한 스크롤로 게시물 로딩 (20개 단위)
- 다른 사용자 프로필 클릭 시 해당 사용자의 게시물 모아보기

### 4-3. 검색 및 팔로우

- 해시태그 또는 닉네임으로 게시물·사용자 검색
- 다른 사용자를 팔로우하면 해당 사용자의 새 게시물이 피드에 우선 노출
- 검색 결과는 게시물 탭과 사용자 탭으로 구분되어 표시됨

---

## 5. 알림

서비스 이벤트와 커뮤니티 활동을 사용자에게 전달하는 기능.

### 5-1. 알림 종류

- 일기 생성 완료 알림
- 캐릭터 생성 완료 알림
- 커뮤니티 알림 (좋아요/댓글/팔로우)
- 서비스 공지 알림

### 5-2. 알림 설정

- 알림 종류별로 수신 여부를 ON/OFF 토글로 설정
- 푸시 알림 / 이메일 알림 채널 선택 가능
- 설정 변경은 즉시 반영됨

---

## 6. 계정 및 인증

사용자 계정 생성·관리·삭제 및 인증을 담당하는 기능.

### 6-1. 회원가입

- 이메일 가입 또는 소셜 로그인(구글/애플/카카오) 지원
- 이메일 가입 시: 이메일, 비밀번호(8자 이상), 닉네임 입력
- 가입 완료 후 이메일 인증 (24시간 내)
- 만 14세 이상만 가입 가능 (생년월일 확인)

### 6-2. 프로필 관리

- 닉네임, 프로필 사진, 자기소개 수정 가능
- 닉네임은 2자 이상 15자 이하
- 비밀번호 변경: 현재 비밀번호 확인 후 신규 입력

### 6-3. 계정 삭제

- 설정 화면에서 "계정 삭제" 진입 → 삭제 사유 선택 → 비밀번호 재확인
- 삭제 시 등록한 반려동물 프로필·캐릭터·일기·게시물 모두 삭제됨
- 삭제 후 30일간 데이터 보관 후 영구 삭제 (30일 내 재로그인 시 복구 가능)
